Что выведет программа? Объяснить вывод программы.

```go
package main

import (
    "fmt"
)

func main() {
    a := [5]int{76, 77, 78, 79, 80}
    var b []int = a[1:4]
    fmt.Println(b)
}
```

Ответ:
```
Создаем массив с помощью литерала, присваиваем переменной "a".
Оператор среза a[1:4] создает новый срез, ссылающийся на элементы
последовательности (нижележащего массива) a со 2-го (с индексом 1) по 4-й (с индексом 3). 
Присваиваем это значение переменной b типа []int. Выводим значение b
в консоль.

нижележащего массива), и происходит перераспределение памяти, все элементы
старого массива копируются в новый, или только те, что были доступны через слайс?
```
